# 答辩前一页速记

## 1. 项目一句话

用 SPY 日线数据完成“清洗 -> 特征 -> ML -> 风险与回测 -> LSTM -> RL 环境 -> Q-learning”的课程级金融数据挖掘实验，重点是方法闭环和客观解释，不构成投资建议。

## 2. 数据对象

| 项目 | 内容 |
|---|---|
| 标的 | SPY，跟踪 S&P 500 的 ETF |
| 来源 | Yahoo Finance，Python `yfinance` |
| 时间 | 2015-01-02 至 2025-12-31 |
| 清洗数据 | 2766 行 x 6 列 |
| 特征数据 | 2707 行 x 15 列 |
| OHLCV | Open、High、Low、Close、Volume，另保留 Adj Close |

## 3. 九项技术特征

```text
return_1d, return_5d
ma_5, ma_10, ma_20, ma_60
volatility_20, volume_ma_20, close_ma20_ratio
```

一句话：收益率描述变化，均线描述趋势，波动率描述风险，成交量均线描述活跃度，价格均线比描述相对位置。

## 4. 模型与结果

| 模型或策略 | 关键结果 | 一句话解释 |
|---|---:|---|
| Logistic Regression | accuracy `0.5886`，recall `1.0000` | 542 个测试样本全部预测上涨，不能只夸 accuracy |
| Random Forest | accuracy `0.4188`，recall `0.0282` | 只预测 14 个上涨日，泛化不足 |
| LSTM | accuracy `0.4540`，AUC `0.4916` | AUC 接近 0.5，短期区分能力弱 |
| Random policy | return `-57.64%` | 随机频繁变仓被成本削弱 |
| Buy & Hold | return `+227.12%` | 研究期 SPY 长期上涨 |
| Q-learning | return `+6.56%` | 正收益但明显弱于买入持有，且不是样本外评价 |

## 5. 风险指标一句话

| 指标 | 一句话 |
|---|---|
| annual return | 日均收益乘 252 的年化估计 |
| annual volatility | 日收益标准差乘 `sqrt(252)` |
| Sharpe ratio | 每单位波动对应多少收益 |
| maximum drawdown | 从历史高点到后续低点的最深跌幅 |
| VaR 95% | 最差约 5% 交易日的分界 |
| CVaR 95% | 进入最差 5% 后的平均损失 |

SPY 最大回撤：`-34.10%`。高累计收益不代表过程低风险。

## 6. 时间对齐必背

```text
监督标签：target_return_1d = return_1d.shift(-1)
Stage 4：position = signal.shift(1)
Stage 6：当前状态选动作，用下一期收益计算 reward
```

一句话：今天生成的信息只能用于下一期，不能偷看未来答案。

## 7. Q-learning 三要素

| 要素 | 本项目 |
|---|---|
| 状态 | `return_1d`、`close_ma20_ratio`、`volatility_20` 各 3 桶，共 27 状态 |
| 动作 | `0=空仓`，`1=持有` |
| 奖励 | 下一期持仓收益减去 0.1% 变仓成本 |

更新公式：

```text
Q(s,a) <- Q(s,a) + alpha * [r + gamma * max Q(s',a') - Q(s,a)]
```

## 8. 五个边界

1. Stage 4 不扣成本；Stage 6 扣 0.1% 变仓成本。
2. Stage 4 `win_rate=1.66%` 是正收益日比例，不是逐笔胜率。
3. RL 在同一整段历史上训练与评价，没有严格样本外测试。
4. LR 输入未标准化，且特征尺度差异较大。
5. 当前结果不构成任何实际投资建议。

## 9. 十个最重要问答

### 1. 为什么选 SPY？

SPY 是有市场代表性的 ETF，数据完整，受单只股票偶发事件影响相对较小。

### 2. 为什么预测方向而不是价格？

方向分类更容易解释，也便于使用分类指标和简化回测。

### 3. 为什么不能随机切分？

随机切分可能让未来数据进入训练集，造成不符合真实预测过程的信息泄漏。

### 4. 为什么 LR 0.5886 不能夸大？

因为它把所有测试样本都预测为上涨，accuracy 主要来自上涨类别占比。

### 5. 为什么 RF 效果差？

它只预测 14 个上涨日，上涨召回率仅 2.82%，当前设定下泛化不足。

### 6. LSTM AUC 0.4916 说明什么？

接近 0.5，当前模型区分涨跌的能力接近随机排序。

### 7. 为什么信号要 shift 一天？

日期 `t` 产生的信号只能用于 `t+1`，否则有未来函数风险。

### 8. 为什么 Q-learning 没超过买入持有？

SPY 长期上涨，且 Q-learning 状态粗、动作简单、训练有限、频繁交易有成本。

### 9. 为什么没有 DQN/PPO？

本项目定位是课程级入门验证，表格型 Q-learning 更适合讲清基本流程。

### 10. 项目最重要结论是什么？

复杂模型不一定更好，金融短期预测很难；项目价值是完整方法验证和风险意识。
