## 1. 监督学习分类指标

### 1.1 基础分类指标

| 指标 | 定义 | 取值范围 | 说明 |
|------|------|----------|------|
| **accuracy** | (TP + TN) / (TP + TN + FP + FN) | [0, 1] | 整体准确率，在类别不平衡时可能产生误导 |
| **balanced_accuracy** | (sensitivity + specificity) / 2 | [0, 1] | 两个类别的平均召回率，对类别不平衡更稳健 |
| **precision** | TP / (TP + FP) | [0, 1] | 预测为正类的样本中实际为正类的比例 |
| **recall** (sensitivity) | TP / (TP + FN) | [0, 1] | 实际为正类的样本中被正确预测的比例 |
| **f1** | 2 × precision × recall / (precision + recall) | [0, 1] | precision 和 recall 的调和平均 |
| **roc_auc** | ROC 曲线下面积 | [0, 1] | 衡量模型对两个类别的区分能力，0.5 表示随机 |

### 1.2 混淆矩阵

| 指标 | 定义 |
|------|------|
| **TN** (True Negative) | 实际为负类且预测为负类的样本数 |
| **FP** (False Positive) | 实际为负类但预测为正类的样本数 |
| **FN** (False Negative) | 实际为正类但预测为负类的样本数 |
| **TP** (True Positive) | 实际为正类且预测为正类的样本数 |

在本项目中，正类（1）表示"下一交易日上涨"，负类（0）表示"下一交易日下跌"。

### 1.3 预测分布指标

| 指标 | 定义 | 说明 |
|------|------|------|
| **positive_rate_pred** | 预测为正类的样本数 / 总样本数 | 模型预测上涨的比例，反映模型是否存在类别偏置 |
| **positive_rate_true** | 实际为正类的样本数 / 总样本数 | 测试集中实际上涨的比例 |

### 1.4 Permutation Importance

| 指标 | 定义 | 说明 |
|------|------|------|
| **importance_mean** | 打乱某特征后模型性能下降的平均值 | 值越大表示特征越重要 |
| **importance_std** | 打乱某特征后模型性能下降的标准差 | 衡量重要性估计的稳定性 |
| **scoring** | 用于评估的评分函数 | 本项目使用 roc_auc，失败时回退到 balanced_accuracy |

注意：Permutation importance 仅作为模型解释辅助，不代表因果关系。

## 2. 回测与风险指标

### 2.1 策略收益指标

| 指标 | 定义 | 说明 |
|------|------|------|
| **cumulative_return** | (最终权益 - 初始权益) / 初始权益 | 策略的总收益 |
| **annual_return** | 日均收益率 × 252 | 年化收益率 |
| **annual_volatility** | 日收益率标准差 × √252 | 年化波动率 |
| **sharpe** (Sharpe Ratio) | annual_return / annual_volatility | 风险调整后收益，假设无风险利率为 0 |
| **max_drawdown** | min((权益 - 历史最高权益) / 历史最高权益) | 最大回撤，取值为负 |
| **calmar** (Calmar Ratio) | annual_return / |max_drawdown| | 收益与最大回撤的比值 |
| **win_rate** | 收益为正的交易日数 / 总交易日数 | 胜率 |
| **trade_count** | 仓位变化次数 | 交易次数 |

### 2.2 市场风险指标

| 指标 | 定义 | 说明 |
|------|------|------|
| **skewness** | 收益率分布的偏度 | 负偏度表示左尾风险较大 |
| **kurtosis** | 收益率分布的峰度 | 高峰度表示极端事件概率较高 |
| **var_95** (VaR) | 收益率分布的 5% 分位数 | 95% 置信水平下的最大损失 |
| **cvar_95** (CVaR) | 收益率 ≤ VaR 的样本均值 | 条件风险价值，即极端损失的期望 |
| **positive_day_ratio** | 收益率 > 0 的天数 / 总天数 | 上涨日占比 |

### 2.3 交易成本处理

本项目中交易成本按以下方式扣除：

```
交易成本 = |position_t - position_{t-1}| × transaction_cost
策略收益 = position × 市场收益 - 交易成本
```

其中 `transaction_cost = 0.001`（即 0.1%）。

| **q_table_states** | Q-table 中被访问的状态数 | 最多 27 个（3×3×3） |

### 3.2 Q-learning 评估指标

| 指标 | 定义 | 说明 |
|------|------|------|
| **total_return** | 测试集上的总收益 | 与回测 cumulative_return 定义一致 |
| **sharpe_ratio** | 测试集上的夏普比率 | 与回测 sharpe 定义一致 |
| **max_drawdown** | 测试集上的最大回撤 | 与回测 max_drawdown 定义一致 |
| **number_of_trades** | 测试集上的交易次数 | 仓位变化次数 |
| **final_equity** | 最终权益值 | 初始为 1.0 |

### 3.3 奖励函数

```
reward = position × market_return - transaction_cost
```

其中：
- `position ∈ {0, 1}`：0 表示空仓，1 表示持有
- `market_return`：下一期市场收益率（已除以 100）
- `transaction_cost`：仅在仓位变化时扣除

| 特征 | 桶 0 | 桶 1 | 桶 2 |

| close_ma20_ratio | < 0.98 | [0.98, 1.02] | > 1.02 |
