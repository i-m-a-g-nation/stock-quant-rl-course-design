## 5. 关键发现

### 5.1 所有模型 R² 为负值

所有回归模型的 R² 均为负值（-0.01 到 -1.09），说明模型预测效果不如直接用训练集均值作为预测值。这是金融收益率回归的典型结果：日收益率噪声极大，线性模型和树模型难以捕捉有效信号。

### 5.2 回归误差看似小但交易意义有限

MAE 约 0.0076~0.0105（约 0.76%~1.05%），RMSE 约 0.011~0.014（约 1.1%~1.4%）。表面看误差不大，但与日收益率的波动幅度（std≈1.13%）相比，误差接近波动本身，说明模型几乎没有预测能力。

### 5.3 预测收益率幅度很小

所有模型的预测收益率均值接近 0（-0.004 到 0.0005），预测值集中在极小范围内波动。DummyRegressor 始终预测训练集均值（约 0.047%），其他模型的预测值也在 0 附近震荡，缺乏区分度。

### 5.4 方向命中率接近随机

- DummyRegressor（始终预测正收益）：55.56%，等于实际上涨比例
- LinearRegression：49.39%，低于 Dummy
- Ridge：49.63%，低于 Dummy
- RandomForestRegressor：50.89%，略高于 Dummy
- GradientBoostingRegressor：48.13%，低于 Dummy

没有模型显著优于 DummyRegressor 的方向命中率。

### 5.5 相关系数接近 0

所有模型的 pearson_corr 和 spearman_corr 均在 0 附近（-0.0003 到 0.056），标准差较大（0.06~0.12），说明预测值与真实值几乎没有线性或单调相关性。

### 5.6 回测结果分析

- DummyRegressor 和 RandomForest 的 total_return 最高（1.59%），但主要源于市场趋势
- LinearRegression 的 total_return 为负（-0.58%），说明线性模型的预测信号在样本外有害
- 所有模型的回测收益均较低，不具备稳定的交易意义

### 5.7 预测 vs 真实散点图

散点图显示预测值高度集中在 0 附近，与真实收益率的分散分布形成鲜明对比。这进一步说明模型未能捕捉收益率的变化模式。

## 6. 不能夸大的结论

1. 所有回归模型 R² 为负，预测效果不如均值基线
2. 预测收益率幅度极小，缺乏区分度
3. 方向命中率接近随机，没有模型显著优于 Dummy
4. 相关系数接近 0，预测值与真实值无显著关联
5. 回测收益较低，不具备稳定的交易意义
6. 回归误差看似小，但与收益率波动相比，预测能力有限
7. 本实验仅用于研究验证，不构成投资建议

## 8. 下一步建议

1. **三分类交易信号** — 引入"震荡/不交易"类别，减少无意义交易
2. **更长预测周期** — 预测 future_5d_return 而非 next_return
3. **特征工程** — 引入宏观、情绪、基本面等外部特征
4. **模型改进** — 尝试更复杂的模型或特征选择
5. **信号强度设计** — 用预测收益率的大小控制仓位比例
